# Inscribing — Documentação do Sistema

Esta documentação descreve as funcionalidades e regras de negócio do Inscribing, uma plataforma web de anotações e gestão de tarefas com chatbot FAQ baseado em busca semântica.

| Módulo | Funcionalidades | Regras |

| [Autenticação e Conta](autenticacao-e-conta/) | Login, cadastro, sessão | RN01-RN03 |
| [Interface e Navegação](interface-e-navegacao/) | Sidebar, breadcrumbs | RN04-RN05 |
| [Gestão de Coleções](gestao-de-colecoes/) | Criação, gerenciamento | RN06-RN08 |
| [Edição de Textos e Blocos](edicao-de-textos-e-blocos/) | Sistema de blocos, listas de tarefas | RN09-RN12 |
| [Conhecimento e Suporte](conhecimento-e-suporte/) | FAQ estático, chatbot | RN13-RN21 |
